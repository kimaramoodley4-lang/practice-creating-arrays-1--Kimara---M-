# Practice: Creating Arrays

## Project Description
This project is a part of the LaunchCode JavaScript curriculum. It demonstrates the fundamental concepts of building, indexing, and logging arrays using JavaScript.

## Tasks Completed
* Created an array of 5 grocery items using array literals.
* Created an array of 3 numbers representing prices.
* Utilized `console.log()` to output entire arrays and specific elements via index.

## Technology Used
* JavaScript (ES6)
* Node.js / Browser Console
